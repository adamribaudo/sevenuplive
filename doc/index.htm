Introduction

7up Live is an application written for the monome and Ableton Live's Max4Live that crams 7 unique pieces of functionality into any number of 8x8 grids. With it, your monome is fully integrated into your Live workflow, allowing you to sequence beats, chop up loops, record melodies, and send control signals. Each 7up component is represented as a device in Live that retains all the Live's native functionality like preset saving/loading and automation.

Requirements

In order to use 7up Live you'll need:

  • A monome (any type will do)
  • Ableton Live 8.1 or greater with Max4Live installed
  • MonomeSerial in order for your monome to comunicate with 7up
  • The Java Runtime Environment 1.5 or greater (most people will have this installed already)

You can download the latest version of 7up here.

In order to install 7up you'll need to follow these instructions:

  1. Unzip the download package to a directory of your choosing
  2. Copy the files in the "Supporting" folder into Max's "Java" folder
    • For Windows this will be C:\Program Files\Cycling '74\Max 5.0\Cycling '74\java\lib
    • For Mac this will be /Applications/Max5/Cycling '74/java/lib

Getting Started

We've provided a sample Ableton Live set here loaded with all of the components, but it's by no means required and we'd suggest you read the instructions below to learn how to make a set from scratch.

  1. Open Monome Serial and note your prefix, host port, and listen port
  2. Open Ableton Live
  3. Use Ableton's file browser to navigate where 7up Live was unpackaged and drag "SevenUp4Live.amxd" into a new track.
    • This is the brain of 7up and should always be loaded first before any other 7up components.
    • Only load one "SevenUp4Live.amxd" into any set.
  4. You should now see the following in your device chain:
  5. Check to see that your prefix, host and listen port are set correctly and press the "Start/Stop Connection" button. If all went well, you should see a blinking "7" appear on your monome!
  6. Next, we'll try using a simple component, the Stepper. The Stepper allows you to create patterns of samples, typically beats. In order to do this, use the file browser to drag "SevenUpStepper.amxd" into a new track.
  7. Next, drag a "Drum Rack" device (in the instruments folder) into the same track so that's it's after the Stepper
  8. Next, drag some samples in the pads labeled C3 to F#3 and press play. You should see a line move vertically down the monome.
